# @Huskie-ui/widget

## Install

```bash
npm i @huskie-ui/widget
# or
yarn add @huskie-ui/widget
```

## Usage

```vue
<template>
 <h-widget/>
</template>

<script lang="ts">
import { defineComponent, toRefs } from 'vue'
import HWidget from '@huskie-ui/widget'

export default defineComponent({
  components: { HWidget },
  setup() {}
})
</script>
```
