# @sum-ui/view

## Install

```bash
npm i @sum-ui/view
# or
yarn add @sum-ui/view
```

## Usage

```vue
<template>
    <sum-view />
</template>

<script lang="ts">
import { defineComponent, toRefs } from 'vue'
import SumView from '@sum-ui/view'

export default defineComponent({
    components: { SumView },
    setup() {}
})
</script>
```
