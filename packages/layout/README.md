# @Huskie-ui/layout

## Install

```bash
npm i @huskie-ui/layout
# or
yarn add @huskie-ui/layout
```

## Usage

```vue
<template>
 <huskie-layout/>
</template>

<script lang="ts">
import { defineComponent, toRefs } from 'vue'
import HuskieLayout from '@huskie-ui/layout'

export default defineComponent({
  components: { HuskieLayout },
  setup() {}
})
</script>
```
